# Quiz System
A Java console-based quiz application with multiple-choice questions and score calculation.

## Features
- MCQ based quiz
- Score calculation
- User interaction through console

## Concepts Used
- Java
- OOP
- Arrays
- Packages
- Methods

## How To Run
1. Open project in VS Code or IntelliJ
2. Compile Java files
3. Run Main.java
