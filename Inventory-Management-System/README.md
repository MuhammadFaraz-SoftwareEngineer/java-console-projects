# Inventory Management System
A basic Java console application for managing inventory and stock records.

## Current Features
- Add products
- View products
- Basic stock management

## Concepts Used
- Java
- OOP
- Arrays
- Packages
- Methods

## Future Improvements
- Product categories
- Advanced search
- File handling

## How To Run
1. Open project in VS Code or IntelliJ
2. Compile Java files
3. Run ManagementSystem.java
