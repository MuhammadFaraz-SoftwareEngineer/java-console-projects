# Java Console Projects
This repository contains multiple Java console-based applications developed using Core Java concepts.

## Projects Included

### 1. Library Management System
A console application to manage library operations like adding books, searching books, issuing and returning books.

### 2. Quiz System
A console-based quiz application with score calculation and multiple-choice questions.

### 3. Inventory Management System
An inventory application for managing products and stock records.

## Concepts Used
- Java
- Object-Oriented Programming (OOP)
- File Handling
- Arrays and Collections
