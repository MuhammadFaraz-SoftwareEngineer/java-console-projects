# Library Management System
A Java console-based application for adding books, users and library issue/return operations.

## Features
- Add books
- Search books
- Add students
- Issue books
- Return book

## Concepts Used
- Java
- OOP
- Arrays
- Methods

## How To Run
1. Open project in VS Code or IntelliJ
2. Compile Java files
3. Run Library.java
